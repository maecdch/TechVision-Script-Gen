# TechVision Script Gen

> 🎥 **Hardcore Technical Video Script Generator**
> Powered by Gemini 2.5 Flash & React

## ⚡ Introduction

TechVision is a specialized tool for technical content creators. It generates high-quality, 8-10 step video scripts in the style of **3Blue1Brown** or **Hardcore Tech** videos.

Input a complex topic (e.g., "Transformer Architecture"), and get a structured timeline with:
- 🇨🇳/🇺🇸 Bilingual Titles
- 📐 Core Mathematical Formulas (LaTeX rendered)
- 🎬 Precise Visual Directives
- 🧠 Concise Explanations

## 🛠 Features

- **AI Director Persona**: Generates scripts with a distinct "hardcore" tone.
- **Visual-First Workflow**: Each step includes specific visual element descriptions (e.g., "Rotating 3D Matrix").
- **Math-Ready**: Built-in KaTeX rendering for complex equations.
- **Cyberpunk UI**: Glassmorphism, neon accents, and grid backgrounds.
- **Export Ready**: One-click JSON copying for your production pipeline.

## 🚀 Getting Started

### Prerequisites
- Node.js 18+
- A Gemini API Key

### Installation

1. **Install Dependencies**
   ```bash
   npm install
   ```

2. **Configure Environment**
   Create a `.env` file in the root directory:
   ```env
   GEMINI_API_KEY="your_api_key_here"
   ```

3. **Run Locally**
   ```bash
   npm run dev
   ```

## 💻 Tech Stack

- **Framework**: React 19 + Vite
- **Styling**: Tailwind CSS v4 + Motion
- **AI**: Google GenAI SDK
- **Math**: KaTeX
